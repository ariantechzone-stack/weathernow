# WeatherNow 🌤

A real-time weather app built with React, Tailwind CSS, and OpenWeatherMap API.

🔗 **[Live Demo](https://weathernow-indol.vercel.app/)** &nbsp;&nbsp; 

## Features
- 🔍 Search any city worldwide
- 🌡 Current weather (temp, humidity, wind, pressure, visibility)
- ⏱ Hourly forecast (next 24 hours)
- 📅 5-day forecast
- 🎨 Dynamic background based on weather condition
- ⚡ Quick city buttons for instant search

## Tech Stack
- React 18
- Tailwind CSS v3
- Vite
- OpenWeatherMap API
- Context API for state management

## Setup

1. Get a free API key from [openweathermap.org](https://openweathermap.org)
2. Open `src/context/WeatherContext.jsx`
3. Replace `YOUR_API_KEY_HERE` with your key
4. Run:

```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → Import to Vercel → Add env variable or paste key directly.
